Jour 9

Après avoir traversé la forêt plongée dans le noir, je suis arrivé devant une étrange structure faite de blocs inconnus.

Devant moi se trouvait un mur rempli de lignes de code. Les caractères bougeaient tout seuls comme si quelqu'un contrôlait cet endroit depuis l'intérieur.

Soudain, un message apparut :

"Système verrouillé. Un mot de passe est nécessaire pour continuer."

Sous le message, deux symboles étaient affichés :

❌ 🩸

Je ne comprenais pas tout de suite ce que ces symboles voulaient dire. Mais en les observant, je compris que le code cachait un indice.

Le mot de passe était lié à quelque chose qui augmente, quelque chose qui avance petit à petit...

Le mot de passe est : croissant
